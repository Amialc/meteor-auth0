meteor-auth0
============================

Meteor's sample (TODO APP) using Auth0 as external login service

### Usage

~~~
git clone https://github.com/auth0/meteor-auth0.git
~~~

> update settings.json with your auth0 Domain, Client ID and Client Secret.

~~~
cd meteor-auth0
meteor
~~~
